# Welcome to daily-journal

A blog website built using Node.js, passport.js and ejs.

You can check out the website on:

## Install

```bash
# install requirements
npm install
```

## Run server

```bash
# run app
node app.js
```

## Open app

Your app will now be accessible at localhost:3000
